Промежууточная контнрольная работа. Часть 2.

Задание:
Необходимо написать программу – розыгрыша игрушек в магазине детских 
товаров. Необходимо применять принциры ООП и работу с файлами.

Программа должна содержать:
- минимум один класс, включающий
id игрушки (будет private),
текстовое название,
количество
частота выпадения игрушки (вес в % от 100)
-метод добавление новых игрушек и возможность изменения веса (частоты выпадения игрушки)
-возможность организовать розыгрыш игрушек.
